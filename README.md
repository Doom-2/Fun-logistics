# Funny logistics
### In-store delivery example

Program run:
* Type `python3 main.py` in the project root

Features:
* remaining stock, storage capacity, excess in the request, product range limit are taken into account
* zero and negative number checks
* checking for merchandise
* keyword spell check

Details:
* An abstract class for basic storage
* Custom Exception Classes
* Getters are used
